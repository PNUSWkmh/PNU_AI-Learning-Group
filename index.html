// 01 ABOUT

프로그램 소개

정보의생명공학대학 재학생들의 AI·SW 전공역량을 심화하고, 협업과 창의적 문제 해결 능력을 강화하는 자율 학습 프로그램입니다.

// TARGET

모집 대상

정보의생명공학대학 SW전공 재학생 (복수전공 포함). 타 전공자도 참여 가능하나 팀장 및 구성원의 50%는 SW전공 학생이어야 합니다.

// PERIOD

운영 기간

2026년 6월 15일(월)부터 8월 31일(월)까지 약 11주간 진행됩니다.

// TEAM

팀 구성

팀장 포함 최소 4명 이상으로 구성. 정컴 20팀, DS 10팀 예상으로 총 30팀을 선발합니다.

// TOPIC

활동 주제

웹·모바일 앱 개발, 데이터분석 프로젝트 수행. 결과물 기반으로 외부 경진대회 참가가 필수입니다.
